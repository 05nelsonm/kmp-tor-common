Lambda for building and/or spawning a process.

e.g. (Using the kmp-process library)

val builder = loaderExec.process(myBinder) { tor, configureEnv ->
    Process.Builder(command = tor.path)
        .args(myTorArgs)
        .environment(configureEnv)
        .stdin(Stdio.Null)
}
